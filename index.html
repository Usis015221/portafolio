Casos de éxitos

Proyecto Aprendiendo Juntos

En el año 2023 mediante la conformación de un equipo de trabajo se desarrollo el sitio web Aprendiendo Juntos, con la finalidad de ser un apoyo para los docentes en la formación académica de los niños en edades de 6 a 8 años, mediante juegos interactivos basados en los contenidos educativos de los docentes.

Proyecto Learning Kids

En el año 2022 para este proyecto se conformo un equipo de trabajo, el cual desarrollo la aplicación móvil llamada Learning Kids, cuya finalidad era ser un apoyo para aquellos niños los cuales se les dificultaba aprender a leer, la aplicación les enseñaba a identificar las letras y numeros mediante imagenes y sonidos, también le enseñaba acerca de los colores para que los niños no se aburrieran de utilizar la aplicación. Para el desarrollo de esta aplicación se utilizo el lenguaje de programación Java y base de datos local llamada SqLite.

Proyecto Child Reading

En el año 2023 para este proyecto se conformo un equipo de trabajo, el cual desarrollo la aplicación móvil llamada Child Reading, el proposito de esta aplicación es que los niños pudieran acceder a libros digitales y así poder desarrollar su pasión por la lectura, también la aplicación contaba con un apartado para que los niños crearan sus propias historias. Para el desarrollo de esta aplicación se utilizo el lenguaje de programación Java y base de datos local llamada SqLite.
